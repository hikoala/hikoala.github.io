Introduction

Welcome to Koala's integration guide!


This guide is there to simplify your understanding of how our API works and make your integration as smooth as possible. By the end of your reading, you will have all the keys to integrate properly with our API and start selling awesome travel insurances.

This document complements our low-level documentation using OpenAPI specification. Inside this document, you will find:

  • An explanation of the vocabulary we use
  • A guide to our APIs core concepts
  • The various possible returns from our APIs

We advise you to first read this document before going into our low-level documentation.
