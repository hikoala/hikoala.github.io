Go into production!

Now that you know everything about our Partner API, it is time for you to validate your integration to receive your production token.

Step 1: Check if your staging token is valid‚Äã

Before doing anything, it is important to check that your staging token is valid or not.

To check your token validity, you should do the following:

curl -X GET "https://staging-partner.api.hikoala.co/quotes" -H "Content-Type: application/JSON" -H "accept: application/JSON" -H "Authorization: Bearer <token>"
note

If you receive an API answer other than 401, you are good to go to Step 2 üëç


Step 2: Send us test Quotes‚Äã

To check that you have implemented the right logic to obtain prices from us, you should post at least 5 valid Quotes.

Inside your payload, you should use the test data of your choice, provided that it respects the data format.

Caution: Please only use dates in the future for your tests, otherwise the API will return you an error message.

If you have questions or doubt about this step, please refer to our Quotes page.

note

Once you have 5 successful quotes, you can go to step 3 üëå


Step 3: Send us test Subscriptions‚Äã

Now that you have obtained successful Quotes, we should turn those Quotes into Subscriptions.

Just as for the Quotes, you should use the test data of your choice inside the payload.

If you have questions or doubt about this step, please refer to our Subscriptions page.

note

Once you have 5 successful subscriptions, you can either:

  • Go to step 4 if you are selling Koala Flex üê®
  • Go to step 5 if you are selling any other Koala product ü§©

Step 4: Send us test claims (only if you sell our Koala Flex)‚Äã

Now that you have created contracts, you should simulate a booking cancellation by the traveler.

Inside your payload, we advice that you use:

  • Fake data for the beneficiaryName
  • An IBAN generated by Random IBAN
  • The following BIC: BNPAFRPP

If you have questions or doubt about this step, please refer to our Contracts page.

note

Once you have 3 successful claims, you can go to step 5 ü§©


Step 5: Change the environment to prod environment‚Äã

Once you have completed all the tasks, please ping us so that we can take a look together.


If everything is functioning correctly, we will send you your prod token and you will be good to go live! üöÄ

Before going live, please change your server URL from https://staging-partner.api.hikoala.co into: https://partner.api.hikoala.co/.
